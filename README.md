# 3MF文件预览工具

一个功能强大的3MF文件预览和处理工具，专为Bambu Lab A1打印机设计。

## 功能特性

1. **3MF文件预览** - 打开和预览3MF文件内容
2. **多打印盘支持** - 识别和处理包含多个打印盘的3MF文件
3. **G-code查看** - 查看文件中的G-code内容
4. **元数据显示** - 显示文件的详细元数据信息
5. **自定义G-code** - 添加自定义G-code代码到打印任务
6. **文件导出** - 导出修改后的3MF文件
7. **换色代码对比** - 分析G-code中的换色代码差异
8. **时间信息提取** - 从G-code中提取打印时间和总估计时间

## 系统要求

- Python 3.x
- PyQt5
- Windows操作系统

## 安装

1. 克隆或下载此项目
2. 安装依赖：
   ```bash
   pip install PyQt5
   ```
3. 运行程序：
   ```bash
   python qt_3mf_previewer.py
   ```

## 使用说明

### 打开文件
- 点击"打开3MF文件..."按钮选择单个3MF文件
- 点击"打开多个3MF文件..."按钮批量打开多个文件

### 查看信息
- 使用下拉菜单切换不同的打印盘
- 左侧显示预览图，右侧显示文件信息

### 添加自定义G-code
- 点击"添加自定义G-code..."按钮
- 可以选择添加到当前文件或所有导入文件
- 支持自定义G-code模板

### 导出文件
- 点击"导出3MF文件..."导出单个文件
- 点击"导出最终文件..."合并多个打印盘并导出

### 换色代码对比
- 点击"换色代码对比"按钮分析G-code中的M620.1代码
- 查看是否存在换色以及具体的换色代码内容

## 技术特性

- 支持多打印盘3MF文件的自动识别和排序
- 智能预览图选择（优先选择非"no light"的预览图）
- 完整的3MF文件结构支持，确保与Bambu Studio兼容
- 时间格式化显示（支持小时、分钟、秒的多种格式）
- 错误处理和用户友好的提示
